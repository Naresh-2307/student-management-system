# Student Management System

A full stack web application to manage student records using React, Flask, and MongoDB.

## Features

- Add new students
- View all students
- Update student details
- Delete students
- Data stored in MongoDB database
- Frontend built with React
- Backend built with Flask API

## Tech Stack

Frontend:
- React
- JavaScript
- HTML
- CSS

Backend:
- Python
- Flask

Database:
- MongoDB

Tools:
- Git
- GitHub
- VS Code

## How to Run

Backend:

```
cd Backend
python app_mongo.py
```

Frontend:

```
cd Frontend/student-frontend
npm install
npm start
```

Backend runs on:
```
http://127.0.0.1:5000
```

Frontend runs on:
```
http://localhost:3000
```
